## ğŸ”— API Integration

The collected sensor data is made available through **APIs** (Application Programming Interfaces).

- APIs act as a bridge between the sensor data servers and the chatbot application
- The chatbot sends user queries (for example: weather conditions, rainfall status, temperature)
- The API fetches the relevant data from the server
- The chatbot processes the response and displays it in a user-friendly format

This approach ensures **accurate, fast, and automated weather information delivery**.
